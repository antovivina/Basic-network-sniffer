# Basic Network Sniffer

## Objective

The objective of this project is to capture and analyze network packets using Python and Scapy.

## Tools Used

* Python
* Scapy
* Npcap (Windows Packet Capture Driver)

## Features

* Captures live network packets.
* Displays source IP address.
* Displays destination IP address.
* Displays protocol information.
* Shows packet summary details.

## Sample Output

Source IP: 192.168.1.8

Destination IP: 192.168.1.1

Protocol: TCP (6)

## Findings

* Network packets travel between devices using source and destination IP addresses.
* TCP packets were observed during web browsing activity.
* Packet sniffing helps understand network communication and troubleshooting.

## Conclusion

This project demonstrates the basic concepts of packet capture and network traffic analysis using Python and Scapy. It provides insight into how data flows through a network and how protocols are used for communication.
